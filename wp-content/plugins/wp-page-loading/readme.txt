== Description ==

WP Page Loading plugin is very helpful for you if you find an animation add before loading your website. Here you can add animation easily before loading page. You can customize the animation and background color.

10+ layouts - Simple, light and great! Add preloader to your website easily, responsive and retina, full customization, compatible with all major browsers.

FEATURES & OPTIONS

* 10+ Layout Presets
* Pure CSS3 animation. Works in any modern browser.
* Unlimited background color. Also can change the opacity.
* Unlimited loader color.
* Strong customize background and color loader.
* Responsive & retina ready.
* Option to enable/disable the preloader.
* Option to enable/disable preloader on-site homepage only.
* Lightweight, fast. Coded with SASS. CSS & JS minified.

== Installation ==

Installation is very easy, you can either upload or install it via WP plugin section (the easy way) or Upload using FTP to wp-content/plugins/

1. Upload the plugin files to the `/wp-content/plugins/wp-loading-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Dashboard > Loading Options screen to configure the plugin
