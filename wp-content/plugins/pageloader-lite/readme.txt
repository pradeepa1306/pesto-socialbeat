== Description ==

PageLoader is a customizable loading screen plugin for WordPress. A quick and lightweight way to make your website much more memorable to your visitors.

**Features:**

* Quick and simple setup
* Color customize every element
* Upload your own loading image or..
* ..use one of 10 included icons and choose from 4 size variations
* Spin or pulsating animation (optionally disable animation altogether)
* Choose from three animation speeds
* When done loading, loading screen can fade or slide away (top/left/right/bottom)
* Change background color and opacity
* Add custom loading text (font size control included)
* Optionally hide on speficied posts/pages
* Optionally show on front page only
* Optionally set custom delay
* Setup and customization can be done quickly and in real-time via the WordPress Customizer

== Installation ==

1. Upload /pageloader-lite/ to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Appearance->Customize->PageLoader Lite" screen to configure and customize the plugin
